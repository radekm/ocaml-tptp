This is the README file for the tptp distribution.

(c) 2012 Radek Micek

TPTP format parsing

tptp is library for parsing FOF and CNF formulas from TPTP format. Library
can parse all TPTP problems which don't contain line starting with "thf(" or
"tff(".

See the files INSTALL.txt for building and installation instructions. See the
file LICENSE for copying conditions.
